  • Панель статуса внизу фиксирует результат последнего действия.

  • Градиентная шапка и карточки метрик отражают «операционный» стиль DataBus.
    • 
                    
  • Тёмные mono-блоки для JSON и событий сокращают визуальный шум.
    • 
                    
  • Для outbox/inbox добавлен быстрый текстовый фильтр по eventId/branchId.
    • 
                    
  • Последняя вкладка сохраняется локально, чтобы ускорить повторные сессии.
